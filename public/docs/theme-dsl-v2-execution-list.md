# Theme DSL V2 落地执行清单（组合函数 + 统一配置项 + 管道写法）

## 目标
- 唯一主写法：`query(...)` + 链式组合 + `|` 管道渲染。
- 静态项唯一入口：`config('key.path', default)`。
- 功能完整覆盖当前系统（列表、分页、搜索、导航、图宣、详情辅助）。
- 安全、性能、统一性优先。

## 执行约束
- 不考虑旧模板兼容与迁移。
- 新语法上线后，文档、示例、校验器全部只保留 V2 写法。
- 禁止“双轨文档”。
